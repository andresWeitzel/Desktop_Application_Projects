##  <img width="35" height="25" src="./doc/assets/gifs/desktop.gif" />  Desktop App

</div>  

</br>


This repository brings together desktop projects built with different approaches and technologies, showcasing solutions with clear interfaces, data persistence, and a modular structure designed for maintenance, scalability, and reuse.
 
 <br>

 * Languages : Java, others.
 * Frameworks : Hibernate, others.
 * Technologies : Java SE, Java EE, others.
 * SGDB : MySQL, PostgreSQL, others.
 * Libraries / Dependencies : mysql-connector, JFreeChart, JCommon,  others.
 * Tools : Maven, Vsc, Git, Postman, Xampp, others.
